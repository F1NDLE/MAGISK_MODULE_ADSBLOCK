# MAGISK_MODULE_ADSBLOCK
Magisk модуль для полной блокировки рекламы, путем замены hosts 


# Плюсы 
> 1.Быстрота установки
> 2.Поддержка всех видов Root прав (Magisk, KernelSU, APatch)

> 3. Полная совместимость с модулем RU_AI_UNLOCKER (by F1NDLE)

# Минусы
Частые обновления hosts. Присутствует OTA

# Обновления (OTA)
Это не то OTA, к которому вы привыкли. OTA встроена в Action. Например, если вы увидели, что модуль нужно обновить, сначала прочитайте описание в Changelog. Если там указано: «Обновлён hosts. Просьба обновить через ACTION» — просто нажмите Action, и модуль сам обновится вместе с hosts, без необходимости перезагружать устройство.
